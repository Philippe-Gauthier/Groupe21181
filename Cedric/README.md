Titre: La routine quotidienne de Cédric 
Chaque matin Cedric se reveille 

1.	Cédric se réveille
	2.	Il regarde l’heure
	3.	Si l’heure est avant 6h, il dort
	4.	Sinon, il continue de dormir
	5.	Si l’heure est entre 6h et 7h, il se lève lentement
	6.	Sinon, il se lève rapidement
	7.	Si le réveil sonne, il l’éteint
	8.	Sinon, il se rendort
	9.	Si c’est un jour d’école, il se prépare
	10.	Sinon, il reste au lit


Matin (branches 11 à 20)
	11.	Si la salle de bain est libre, il prend sa douche
	12.	Sinon, il attend
	13.	Si l’eau est chaude, il se douche
	14.	Sinon, il attends
	15.	Si le dentifrice est disponible, il se brosse les dents
	16.	Sinon, il cherche un autre
	17.	Si il a faim, il mange
	18.	Sinon,il ne mange pas
	19.	Si il est en retard, il mange vite
	20.	Sinon, il mange tranquillement


Transport (branches 21 à 30)
	21.	Si il neige, il prend un manteau
	22.	Sinon, il sort sans manteau
    23.	Si le bus arrive à l’heure, il monte
	24.	Sinon, il attend
    27.	Si il a sa carte STS, il valide
	28.	Sinon, il ne peut pas entrer
	25.	Si le bus est plein, il reste debout
	26.	Sinon, il s’assoit
	29.	Si il est à l’heure, il va en classe
	30.	Sinon, il arrive en retard


École (branches 31 à 40)
	31.	Si le professeur est présent, le cours commence
	32.	Sinon, le cours est annulé
	33.	Si il comprend la matière, il continue
	34.	Sinon, il pose une question
	35.	Si il a un examen, il se concentre
	36.	Sinon, il écoute
	37.	Si il a fini ses cours, il rentre
	38.	Sinon, il reste
	39.	Si il est fatigué, il se repose
	40.	Sinon, il étudie


Soir (branches 41 à 51)
	41.	Si il est après 17h30, il rentre à la maison
	42.	Sinon, il reste dehors
	43.	Si il a des devoirs, il révise
	44.	Sinon → il se repose
	45.	Si il est 19h, il étudie
	46.	Sinon → il attend
	47.	Si il est 20h30, il mange
	48.	Sinon → il continue
	49.	Si il est fatigué, il se couche
	50.	Sinon, il reste éveillé
	51.	Si il est 21h30,la journée se termine
