Titre: jeu de quiz educatif 

# But du programme 
Le but de ce programme est de créer un jeu de quiz educatif. Le joueur doit répondre à 50 questions ou il y a trois (3) niveaux: 
  -Niveau 1: Capitale des pays 
  -Niveau 2: Mathématique et physique 
  -Niveau 3: Culture générale

# Entrées 
   Les entrées du programme sont: 
  -Les réponses du joueur saisies au clavier 
  -le joueur entre 1, 2 ou 3 pour chaque question
  -Les entrées seront récupérées à l'aide de la fonction input() 

# Sorties 
  -Affichage des questions 
  -Affichage des choix de réponse (1,2,3)
  -Les sorties seront affiché a l'écran avec la fonction print()

# Structure de controle 
  Le programme utilise une structure conditionnelle if/else pour vérifier les reponses de l'utilisateur. 
  Dans la fonction reponse()
  -le programme demande une réponse à l'utilisateur avec input()
  -la condition if decision == vrai_réponse vérifie si la réponse saisie correspond à la   bonne réponse 
  -si la condition est vraie, le programme affiche "bonne réponse"
  -sinon le programme affiche "mauvais réponse"
