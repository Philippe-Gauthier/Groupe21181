## Input
Pour commencer, cliquer sur **jouer.bat**. 
Il vous faudra écrire votre choix de réponse dans le terminal. des suggestions vous seront présenté.

## output
suite a vos réponses, du texte apparaitra dans le terminal vous offrant de nouveaux choix. 

## Structure
Chaque choix aura des conséquences sur les futurs choix auquel vous ferez face. plusieurs fin différentes sont obtenable. dans le cas d'une fin insatisfesante, il sera possible de revenir au début de l'arbre des possibles.
