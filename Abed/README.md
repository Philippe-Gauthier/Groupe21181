# Description du projet
le but de ce projet est de créer un programme que tu peut interragir avec en arbre.
Le personnage va faire face a des question, chacune de ces questions auront plusieurs choix.
Selon les choix , le programe pousse le personnage vers différentes branches de l'arbre, qui menent à de nouvelles questions ou à une fin.

# FONCTIONNALITÉ
-Le code affiche une histoire basée sur les décisions de Bobby
-L'histoire s’adapte selon le chemin choisit
-Le programme pose des questions à chaque étape qui sont importante dans l'histoire
-Plusieurs choix sont donné au personnage
-Chaque choix amène à une conséquence différente


# Histoire
Bobby s’en rend compte trop tard.

Le sentier qu’il suivait depuis le début de l’après-midi a disparu. Les marques rouges peintes sur les arbres se sont faites rares, puis inexistantes. Autour de lui, la forêt semble plus dense, plus sombre, comme si elle se refermait lentement.
#
Bobby s’arrête.
Le silence est étrange, pesant.

Il sort son téléphone. Aucun signal. La batterie clignote, presque vide. Un soupir lui échappe tandis qu’il range l’appareil dans sa poche. Il tente de se rappeler le chemin emprunté, mais chaque arbre se ressemble maintenant. Chaque direction semble être la mauvaise.

Le vent fait craquer les branches au-dessus de sa tête. Ce bruit, pourtant banal, lui donne des frissons. La peur s’installe doucement, pas brutalement, mais assez pour faire battre son cœur plus vite.

Dans son sac, Bobby ne trouve pas grand-chose : une bouteille d’eau à moitié vide, un manteau léger et un vieux couteau de poche. Rien qui puisse vraiment le rassurer.

La lumière baisse rapidement.
La nuit approche.

Bobby sait qu’il ne peut pas se permettre de paniquer. Chaque décision aura un impact. Avancer sans réfléchir pourrait l’épuiser. Rester immobile trop longtemps pourrait l’exposer au froid. Chercher un abri pourrait être une chance… ou un risque.

Il passe une main sur son visage, tente de se calmer. La forêt ne lui donne aucun indice. Aucun chemin évident. Aucun bruit humain.

Seulement cette impression troublante d’être observé.

Bobby inspire profondément.

Il est perdu.
Et maintenant, tout dépend de ce qu’il décidera de faire.
